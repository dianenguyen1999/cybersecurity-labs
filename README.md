# Cybersecurity Labs

This repository contains my hands-on cybersecurity practice, including TryHackMe labs and CTF challenges.

## 🔐 Topics Covered
- Log Analysis
- Network Security Fundamentals
- Threat Detection
- Basic Incident Investigation
- Access Control Concepts

## 📁 Structure
- `network-fundamentals.md` → Notes on networking concepts (TCP/IP, DNS, etc.)
- `tryhackme-notes.md` → General cybersecurity lab notes
- `ctf-notes.md` → CTF challenge writeups and approaches

## 🎯 Goal
To build practical cybersecurity skills through hands-on labs and real-world scenarios, focusing on analysis, investigation, and problem-solving.
